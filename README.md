CodeAlpha Data Visualization Task
Project Title

Student Data Visualization using Excel

Description

This project is part of the CodeAlpha Data Analytics Internship. The goal of this task is to transform raw data into visual formats such as charts and graphs to make the data easier to understand.

In this project, student data containing Name, Subject, Marks, Department, Year, and Grade was used. A bar chart was created to visualize student marks and analyze their performance.

Tools Used

Microsoft Excel

Dataset Information

The dataset contains:

30 rows of student data

6 columns:

Name

Subject

Marks

Department

Year

Grade

Visualization Created

Bar Chart showing student marks

Steps Performed

Created student dataset in Excel

Selected Name and Marks columns

Inserted Bar Chart using Excel

Saved the chart as image

Saved the Excel file

Outcome

The visualization helps to easily compare student performance and understand the data clearly. This task improved my understanding of data visualization concepts.
